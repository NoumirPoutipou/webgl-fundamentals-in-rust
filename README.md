# webgl-fundamentals.org examples in rust

Live examples can be seen here : https://noumirpoutipou.github.io/webgl-fundamentals-in-rust/


## Dependencies

Using :
* [cargo web](https://github.com/koute/cargo-web)
* [stdweb](https://github.com/koute/stdweb)


## How to compile

In each directory :

```
cargo web start
```
